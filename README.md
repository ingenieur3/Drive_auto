# Drive_auto 
<p>Dans le cadre de la finalisation du semestre 6, nous avons été amenés à réaliser un projet en binôme dans le cadre du projet DEEP. Ce projet a pour objectif de mobiliser et de mettre en pratique nos compétences en électronique, en informatique ainsi qu’en gestion de projet.</p>
<p>Le projet DEEP se déroule en deux phases. Dans un premier temps, l’accent est mis sur la partie matérielle (hardware), puis dans un second temps sur la partie logicielle, centrée principalement sur le développement du code destiné à la cible.</p>
